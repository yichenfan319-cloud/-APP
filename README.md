##  项目简介

本项目是一款专为武夷学院学生设计的课表查询 Android 应用，解决教务系统无移动端、重复登录等问题。

### 核心功能

-  **自动登录**：一次输入，后续自动登录
-  **记住密码**：本地加密存储，安全便捷
-  **今日课表**：当天课程一目了然
-  **本周课表**：表格形式展示整周课程
-  **实时天气**：显示本地天气信息
-  **每日寄语**：励志语句每日更新
-  **系统公告**：接收服务器推送通知
-  **版本更新**：自动检查新版本
